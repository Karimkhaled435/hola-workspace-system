# HOLA Workspace System 🚀

A smart and modern workspace management system designed to handle client sessions, WiFi voucher access, and real-time operations using Firebase.

---

## 🔒 Private Project Notice

This project is private and proprietary.
Unauthorized use, copying, or distribution of any part of this code is strictly prohibited.

---

## ✨ Features

* 🔐 Secure client login system
* 🎫 WiFi voucher / card system
* ⏱️ Real-time session tracking
* 📊 Admin dashboard with full control
* 🧾 Invoice generation and printing (PDF optimized)
* 🔄 Firebase Firestore integration
* 📍 Location-based validation (anti-fake GPS logic)
* 💬 Live chat between client and admin
* 🎯 Loyalty system (stamps & rewards)

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Tailwind + Custom Styles)
* Vanilla JavaScript (Modular Structure)
* Firebase (Firestore)
* Netlify (Deployment)

---

## ⚙️ System Architecture

* Frontend: Static web app (Netlify)
* Backend: Firebase Firestore
* Authentication: Custom logic (phone-based)
* Data Flow: Real-time updates using Firestore listeners

---

## 🚀 Deployment

The project is deployed using Netlify.

To deploy manually:

1. Upload the project folder to Netlify
2. Ensure all files are properly linked
3. Firebase configuration is correctly set

---

## ⚠️ Security Notes

* Firebase rules must be configured properly
* Sensitive data should not be exposed
* API keys are restricted via Firebase settings
